# Tarea-3
Ejercicio 1: Clase Mascota

Se creó una clase llamada Mascota con los atributos nombre y edad, definidos como privados.
Se implementó un constructor para inicializar los datos, un método mostrar() para visualizar la información y un método setEdad() para modificar la edad.
Desde la clase principal se crea un objeto, se muestran sus datos, se modifica la edad y se vuelven a mostrar los resultados.

Ejercicio 2: Clase Celular

Se creó una clase llamada Celular con los atributos marca y bateria, también definidos como privados.
Se implementó un constructor para asignar valores iniciales, un método mostrar() para ver la información y un método cargar() para aumentar el nivel de batería.
En la clase principal se crea un objeto, se muestra su estado inicial, se realiza una carga y se vuelve a mostrar la información actualizada.
